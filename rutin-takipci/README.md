# 🎯 Rutin Takipçi

Alışkanlıklarını takip et, XP kazan, seviye atla.

**Canlı Demo:** https://rutin-takipci.onrender.com

---

## 🚀 Özellikler

- ✅ Günlük alışkanlık takibi
- ⚡ XP / Seviye sistemi
- 🔥 Streak takibi
- 🎯 Haftalık hedef belirleme
- 🍅 Pomodoro zamanlayıcı
- 📊 İstatistik ve haftalık rapor
- 🏆 Arkadaş liderboardu
- 🎵 Odak müziği (Web Audio API)
- ✨ AI alışkanlık önerisi (Claude API)
- 🎨 5 tema seçeneği
- 📱 PWA desteği
- ❄️ Streak freeze sistemi

## 🛠 Teknolojiler

- **Backend:** Python, Flask, SQLAlchemy
- **Database:** PostgreSQL (Render)
- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS
- **AI:** Anthropic Claude API
- **Deploy:** Render.com

## ⚙️ Kurulum
```bash
git clone https://github.com/aslan0438/rutin-takipci.git
cd rutin-takipci/rutin-takipci
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

`.env` dosyası oluştur:
```
SECRET_KEY=your-secret-key
ANTHROPIC_API_KEY=your-api-key
```
